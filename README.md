# find-emoji
 in this app, you can easily find your emoji name.
 
# how to use:
 
 In text box  section when you put the emoji instant get your emoji name,  or you can click on emoji we have section emoji,  you will be got the emoji name.
  

# Technologies used:

   * HTML
   * CSS
   * Javascript
   * Reactjs
   * Netlify(for hosting)
